## 🤖 AI技术栈

### 核心框架
- **RAG框架**: LightRAG - 知识图谱构建和语义检索
- **Agent框架**: CrewAI - 多Agent协作系统
- **工作流编排**: LangGraph - 复杂AI工作流管理
- **向量数据库**: Qdrant - 语义搜索和相似度匹配

### LLM集成
- **OpenAI**: GPT-3.5/4 API
- **本地模型**: Ollama支持
- **多模型支持**: 兼容OpenAI格式的API
