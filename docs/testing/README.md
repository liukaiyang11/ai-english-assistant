## 测试工具推荐

### 功能测试
- **Web测试**: Selenium, Cypress, Playwright
- **API测试**: Postman, Insomnia, pytest
- **移动端测试**: Appium, Detox

### 性能测试
- **负载测试**: JMeter, Locust, Artillery
- **前端性能**: Lighthouse, WebPageTest

### 自动化测试
- **Python**: pytest, unittest
- **JavaScript**: Jest, Mocha
- **CI/CD**: GitHub Actions, Jenkins
