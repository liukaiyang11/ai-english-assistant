## 🛠 技术栈

### 浏览器插件
- **框架**: Manifest V3 + React + TypeScript
- **构建**: Vite + CRXJS
- **UI**: Ant Design + Tailwind CSS
- **状态管理**: Zustand

### 桌面应用
- **框架**: Electron + React + TypeScript
- **打包**: Electron Builder
- **更新**: Electron Updater

### 移动端
- **技术**: PWA + React + TypeScript
- **UI**: Ant Design Mobile
